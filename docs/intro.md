# Protostar CLI

`protostar` is a self-contained command-line binary — no .NET runtime needed to
run it. It is the first component of [Protostar](https://github.com/voidprojectssoftware/protostar),
built incrementally one ticket at a time.

```console
$ protostar
protostar v0.1.0
Live, continuous refinement of agent skills.

Run protostar --help to see available commands.

$ protostar --version
0.1.0
```

Start with [Installation](./installation.md), or jump to
[Channels](./channels.md) to choose between the stable and edge tracks.
